Java简介

Java
  • basics

什么是Java语言

  • Java是一门高级的面向对象的程序设计语言
  • 1995年由sun公司推出的极富创造力的语言
  • 由Java之父 詹姆斯 戈士林设计，他完成了Java技术的原始编译器和虚拟机
  • Java是一种通过解释方式来执行的语言，规则与C++类似
  • 与C++相比，Java语言简洁可靠，具有较高的安全性

Java的应用领域

  • 桌面应用系统开发
  • 嵌入式系统开发
  • 电子商务应用
  • 企业级应用开发
  • 交互式系统开发
  • 多媒体体统开发
  • 分布式系统开发
  • Web应用系统开发

Java的版本

  • Java SE：标准版，用于桌面应用程序的开发，同时也是Java的基础

  • Java EE：企业版，开发企业环境下的应用程序，针对web开发

  • Java ME：小型版，应用于嵌入式系统的开发

Java API文档

  • API：Application Programming Interface 程序编程接口
  • 包括类的继承结构、成员变量、成员方法、构造方法、静态成员的描述信息和详细说明等等
  • JDK 10 链接：https://docs.oracle.com/javase/10/docs/api/overview-summary.html

Java语言的特性

  1. 简单：语法简单明了，易掌握，提供了丰富的类库、API文档和第三方开发包

  2. 面向对象：Java提倡万物皆对象，语法中不能在类外面定义单独的数据和函数

    ​ 即：Java语言最外部的数据类型是对象，所有的元素都要通过类和对象来访问

  3. 跨平台性：一次编译到处运行，前提：操作系统安装了与系统相关的Jvm虚拟机

    ​ (java虚拟机：用于解释并执行java程序的一个应用软件)

  4. 分布性：操作分布(在多台主机布置相关操作)、数据分布(将数据分别存放到不同的主机上)

  5. 解释型：虚拟机Jvm为Java提供了一个解释环境，让Java代码可以运行

  6. 安全性：没有指针(代码安全验证)、没有内存管理(垃圾回收机制)

  7. 健壮性：Java会检查程序在编译和运行时的错误，并消除错误

  8. 多线程：同一时间执行多项任务，可以进行良好的交互能力和实时行为

  9. 高性能：编译后的字节码文件在解释器中运行，速度较快

  10. 动态：Java可动态的调整库中方法和增加变量，客户端不需任何更改

    ​ 在Java中进行动态调整非常简单和直接

注： (1)垃圾回收机制(不需要收到释放内存)： gc：垃圾回收器

​             java中的一个线程，这个线程有自己的算法控制，我们不能干预

​             我们调用垃圾回收器时它不一定工作

​             代码要符合JVM规范

​             类型的转换要正确
