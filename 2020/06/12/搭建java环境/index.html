搭建java环境

Java
  • basics

  1. JRE(Java Runtime Environment Java运行环境)

    ​ Jre包括Java虚拟机和Java程序所需的核心类库 (Jre = Jvm + 类库)

    ​ 如果要运行一个开发好的Java程序，计算机中只需安装Jre即可

  2. JDK(Java Development kit Java开发工具包)

    ​ Jdk提供给Java开发人员使用

    ​ Jdk其中包含了Java的开发工具，也包含了Jre(Jdk = jre + 开发工具)

    ​ 安装好了Jdk，就不需要在单独安装Jre了

    ​ 其中开发工具举例：编译工具(javac.exe)、打包工具(jar.exe)…

  3. JDK的下载

    ​ 下载网址：http://www.oracle.com/technetwork/java/javase/downloads/index.html
