记录通过阿里云服务器 搭建wordpress个人博客

写在前面

在成功搭建完静态博客以后，使用了一段时间，但是最近发现访问有时候访问不了，需要依靠科学上网才可以访问，于是就买了一台服务器，准备搭建我的博客2.0

这次使用的是lamp进行搭建，那什么是lamp呢，就是Linux、Apache、MySql、PHP，当然还有lnmpa，lnmp，其中n就表示Nginx。

我们的环境是

服务器：CentOS 7.3

博客部署服务器：Apache

后台语言：PHP

数据库：MySql

前端框架：WordPress

伴随以后的学习的深入，可以适当进行重构，不过这都是后话了。

1. 设置www用户

使用命令adduser www创建www用户

使用命令passwd www设置www用户的密码

使用命令chmod -v u+w /etc/sudoers将sudoers文件的权限修改成可编辑

编辑文档vim /etc/sudoers

找到root ALL=(ALL) ALL在下面添加www ALL=(ALL) ALL

使用命令chmod -v u-w /etc/sudoers将sudoers文件的权限修改成不可编辑

2.装apache、MySQL

在终端中输入yum install httpd mariadb-server php php-mysql -y

安装好以后我们进行启动过服务和设置开机启动

由于CentOS7安全性问题，需要手工关闭一下防火墙,并关闭开机启动防火墙

执行完以上两步可以使用ip地址在浏览器中进行访问，可以出现如下图片就可以验证Apache是否安装成功了，如果无法访问可以去控制台看一下http的端口号是否开放。

2.MySql数据库配置

  1. 在上面我们已经安装了MySql数据库，下面我们进行初始化

    使用命令mysql_secure_installation，回车，输入y为root用户设置密码，输入自己的秘密即可

    验证完你输入的密码后会询问你

    • 是否删除系统自带匿名用户

    • 是否不允许远程根登录

    • 是否删除测试数据库

    • 是否现在重新加载权限

    我选的都是是，这个看自己的选择。

  2. 使用命令mysql -u root -p并输入密码进行登录服务器

    我们创建一个数据库wordpress，这里名字可以自定义，但是需要记住这个名字，我们一会儿还要使用它。

    执行完成后我们使用命令exit推出数据库

3.安装wordpress

  1. 下载

    • 方法一：我们可以去wordpress官网下载安装包然后传到服务器上进行安装
    • 方法二：使用命令wget https://cn.wordpress.org/wordpress-4.8.1-zh_CN.tar.gz，进行下载

    但是最近中国服务器访问老是出现出429 ，所以搜索一下国内资源，或者采用科学上网，进行下载了

  2. 上传服务器(使用命令安装的可以忽略)上传到www的家目录下

    以下方法推荐，但是不局限于此，有些高端操作咋也不会呀—_—

    • window：可以使用rz命令，先使用命令yum install lrzsz在服务器上进行安装，然后在shell工具上输入rz回车，就可以选择要上传的东西了，下载的话可以使用sz filename，就可以下载了

    • mac/linux：可以使用rsync命令，先使用yum install rsync进行安装，需要保证两端都有rsync，然后是格式rsync -av 路径 用户名@ip地址:路径（注意：这个操作可以在要上传的服务器或者是本地都可以执行，用户名和ip填写另一台的，需要密码验证）这样即可拷贝

  3. 解压这里解压当前家目录下

    去你存放Wordpress安装包目录下执行tar -zxf wordpress-5.3.2-zh_CN.tar.gz -C /var/www/html/tar -zxf wordpress-5.4-zh_CN.tar.gz

    这时在/var/www/html/目录下有一个wordpress文件夹

    重点：

    在这里的操作的决定你以后的访问路径

    • 如果你想通过ip或者域名直接访问那么你需要移动wordpress文件夹里面的东西到html文件夹下
      • mv /var/www/html/wordpress/* /var/www/html/
      • rm -r /var/www/html/wordpress
    • 如果想通过ip/xxxx或者域名/xxxx那么需要你讲文件夹重命名为xxxx
      • 命令：mv wordpress xxxx

    使用命令mv wordpress htdocs将wordpress改名为htdocs

    然后使用root用户使用命令vim /etc/httpd/conf/httpd.conf修改文件

    1. User 和Group 将apache改为www
    2. vim中使用:%s/\/var\/www\/html/\/home\/www\/htdocs/g将/var/www/html修改为/home/www/htdocs
    3. vim中使用:%s/\/var\/www/\/home\/www/g
    4. 保存

    我是要直接通过ip或者域名访问，所以我按照第一种方法

  4. 配置

    先使用命令在Wordpress的根目录下创建配置文件cp wp-config-sample.php wp-config.php

    编辑wordpress的配置文件vi wp-config.php

    配置完保存，然后我们就可以通过ip地址访问我们的设置界面了

    如果这里出现HTTP ERROR 500，可以参考我的另一篇博客··············

    如果出现Your server is running PHP version 5.4.16 but WordPress 5.3.2 requires at least 5.6.20，可以参考我的另一篇博客············

    这是可以看见界面

    注册好后登录就行了，登陆后就是网站的后台，可以自己进行设定

    这样一个动态博客就搭建好了，以后有时间会持续更新，如何配置，优化博客
